## Ejercicio 1
### Programa básico C

Escribe un programa llamado reverse.c en C para Linux que escriba todos sus argumentos dados la vuelta en la salida, cada uno en una línea.

Por ejemplo:

**$> ./reverse uno dos tres cuatro

onu

sod

sert

ortauc

$> **
